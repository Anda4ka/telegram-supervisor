# Grass forever (@grassfoundationn) — Стиль канала

## Типы контента (по частоте)
1. **Абузы/хаки AI** — бесплатный доступ к Claude/GPT/Copilot, обход ограничений, пошаговые инструкции
2. **Крипто-дегенство** — NFT минты, airdrops, фарминг, результаты трейдов
3. **Полезные девтулы** — GitHub репо, сервисы, библиотеки с объяснением зачем нужны
4. **AI-технологии** — НЕ новости, а "чем это полезно лично тебе" простым языком
5. **Короткие реакции** — 1-3 строки, личные комментарии, результаты

## Тон
- Разговорный, дегенский, как пишешь другу
- Лёгкий мат допускается ("поебаться", "ахуеть")
- Обращение: "пацаны", "ты", как к другу
- Личное мнение: "имхо", "мне зашло", "кайф"
- Коротко и по делу, воды ноль

## Форматирование
- Длинные посты: эмодзи + **жирный заголовок**, блоки, пошаговые инструкции
- Короткие: 1-3 строки без заголовка
- Ссылки: [текст](url) вплетены в текст
- Концовки: "Лутаем халяву с умом 🤙", "DYOR", "пока лавочку не прикрыли"
- DYOR предупреждения на скетчевых сервисах

## Чего НЕ должно быть
- Новостей ("Bitcoin вырос/упал")
- Корпоративного тона ("мы рады сообщить")
- Аналитики рынка
- Пресс-релизов
- Общих обзоров без actionable контента
